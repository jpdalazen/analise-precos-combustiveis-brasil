# Análise dos Preços da Gasolina no Brasil

Este projeto realiza uma análise exploratória dos preços da gasolina no Brasil,
utilizando dados públicos e a linguagem Python.

## Objetivo
Investigar a variação dos preços da gasolina ao longo do tempo, entre estados,
regiões e tipos de combustível, identificando padrões e diferenças relevantes.

## Dados
- Fonte: Dados públicos da ANP
- Formato: TSV
- Total de registros: ~120 mil linhas

## Tecnologias Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Etapas da Análise
- Leitura e tratamento dos dados
- Análise estatística descritiva
- Comparação de preços por estado e região
- Evolução temporal dos preços
- Diferença entre preço de distribuição e revenda

## Estrutura do Projeto
├── dados/
│   └── dadosgasolina.tsv
├── analise_precos_combustiveis_brasil.ipynb
├── README.md

## Principais Insights
- Estados da região Norte apresentaram os maiores preços médios de revenda.
- Há diferença significativa entre preços de distribuição e revenda.
- Observa-se tendência de aumento nos preços ao longo do período analisado.
